# BitHeroesInternal

A Internal mod made for educational purposes, inject using [SharpMonoInjector](https://www.unknowncheats.me/forum/downloads.php?do=file&id=34970) like so <br> 
```smi.exe inject -p "Bit Heroes" -a "C:\path-to-dll\BitHeroesInternal\bin\Release\BitHeroesInternal.dll" -n BitHeroesInternal -c Loader -m Load``` <br> <br>
<br>
Change the dll path to where you downloaded the DLL <br>
To open menu ingame press <b>F8</b>
