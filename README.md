# BitHeroesInternal

A Internal mod made for educational purposes, inject using [SharpMonoInjector](https://www.unknowncheats.me/forum/downloads.php?do=file&id=34970) like so 
smi.exe inject -p "Bit Heroes" -a "C:\path-to-dll\BitHeroesInternal\bin\Release\BitHeroesInternal.dll" -n BitHeroesInternal -c Loader -m Load
Change the dll path to where you downloaded the DLL
To open menu ingame press F8
